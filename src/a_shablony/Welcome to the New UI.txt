How To Enable IntelliJ IDEA NEW UI | | 2022.3 Version

1. Main Menu
   Click the hamburger icon or press Alt \ to access the main menu
2. popup - Всплывающее окно или поп-ап — окно
3. Git Widget
   Switch between branches, create new ones, and access a list of popular Git actions 
   in the popup menu.
4. Settings Sync...
   Run Widget
   Create or edit run/debug configuration 71. Start a run configuration from 
   the dropdown or from the toolbar.
5. Code With Me
   Work remotely on a shared project with your teammates. Review code, edit it 
   together and see changes appear in real time.
6. Tool Windows Layout
   Tool windows are docked to the sides of the main window, with layout options 
   of vertical, horizontal, or split depending on the icon location.
7. More Tool Windows
   Access other available tools via the More Tool Windows icon.
8. Navigation Bar
   See the path to the selected file. To move the bar to the top of the window, 
   in the main menu, select View | Appearance | Navigation Bar.
